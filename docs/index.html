Equation

a set of parameters representing proportions for a Struct

uses a generalized form that supports both Line and Circle structs

d x^2 + e x y + f y^2 + a x + b y = c

  • ensure that the sign of c is passed in correctly
  • can support other conic 2d forms in the future

the parameters are set as strings representing algebraic values
these parameters are held as properties of an object so that the properties can be iterated through for analysis

Algebrite library http://algebrite.org provides all the calculations of the algebraic values and conversion to float when necessary

  • TODO: Equation should be treated as a static immutable object
  • TODO: all values that don't change should be calculated on construction.
new Equation(d: string, e: string, f: string, a: string, b: string, c: string)
Parameters
d (string) coefficient for x^2 term - always 1 for a circle - 0 for line
e (string) coefficient for xy term - should always be zero in this model
f (string) coefficient for y^2 term - always 1 for a circle - 0 for line
a (string) coefficient for x term
b (string) coefficient for y term
c (string) constant - WATCH SIGN - note that we position the c value on the other side of the equations

Explorer

provides the context for an algebriac framework to explore proportions in classical constructive geometry.

  • establishes a two dimensional cartesian plane
  • maintains the Model - a collection of Element objects
  • controls the View - methods for rendering an Element in SVG and UI lists
  • provides an interface to create Point and Struct objects
  • script the construction in main
new Explorer()
Example
const E = new Explorer();

//initial points set by X Y
let P0 = new Point( "-1/2", "0" )
E.addPoint(P0)
P0.log()

let P1 = new Point( "1/2", "0" )
E.addPoint(P1)
P1.log()

const L1 = new Line( P0, P1 )
E.addStruct(L1)
L1.log()

Point

represents an ordered List of two algebraic values

  • extends from Element
  • can be considered the coordinates x, y within a two dimensional cartesian plane
  • other than the two "given" starting points, new points in the model are derived from the root set of a System of algebraic expressions defined by two Struct
  • the starting values of the first two points are explicitly set and have no parent Struct
